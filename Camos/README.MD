# Costume Camos Costume Pack
![Pics](https://github.com/darmuh/YapYap-CostumeMods/blob/master/Camos/pictures.png?raw=true)  

## Overview
This Costume texture pack adds 6 new Camo-Style Costume textures to choose from!  
	- Blue Tiger Camo  
	- Classic Camo  
	- Desert Camo  
	- Digital Camo  
	- Red Tiger Camo  
	- Woodland Camo  

## Who needs it?
- All players who want to see the new costume colors need this mod installed.  

## Dependencies
This pack uses the following library:  
* **More Colors** (`Omniscye-More_Colors-1.0.4`)
