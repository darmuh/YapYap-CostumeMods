# Costume Camos Costume Pack
![Pics](https://github.com/darmuh/YapYap-CostumeMods/blob/master/Camos/pictures.png?raw=true)  

## Overview
This Costume texture pack adds 6 new Camo-Style Costume textures to choose from!  
	- Blue Tiger Camo  
	- Classic Camo  
	- Desert Camo  
	- Digital Camo  
	- Red Tiger Camo  
	- Woodland Camo  

## How to use
Install this, and go into the game to change your colors and you will see the new costumes installed!
- This mod uses the More Colors library by Omniscye, so you will get 3 other costumes added with it as that will be installed auto if using a mod manager.

## Who needs it?
- All players who want to see the new costume colors need this mod installed.

## Dependencies
This pack uses the following library:
* **More Colors** (`Omniscye-More_Colors-1.0.4`)
